### Node.js

- Is a JavaScript runtime
- Great for handling a lot of concurrent requests
- Bad for CPU intensive work
- Rich Open Source ecosystem

---

### NPM

Node Package Manager

- Directory of packages
- CLI (Command Line Interface)
- Install/Update dependencies

---

### Express

Minimalist web framework for Node.js

- Simplifies common server tasks
  - Sockets
  - Routes
  - Requests and responses
  - Error handling
  - etc
- Open Source, great community support
